SN Admin - Organigramme Administration Sénégalaise

Gestion complète de l'organigramme de l'administration publique

Fonctionnalités

Gestion Back-office

  • Gestion des ministères et institutions
  • Gestion des directions générales et régionales
  • Gestion des services, bureaux et cellules
  • Gestion des agents publics avec coordonnées
  • Tableau de bord statistiques
  • Rapports PDF (organigramme, annuaire)

Portail Public

  • Page publique navigable
  • Recherche avancée d'interlocuteurs
  • Affichage hiérarchique
  • Responsive (mobile, tablette, desktop)
  • API publique (optionnel)
  • Accessible WCAG 2.1 AA

Installation

1. Télécharger et installer le module
2. Redémarrer Odoo
3. Aller dans Apps et rechercher "SN Admin"
4. Cliquer sur "Installer"
5. Importer les données depuis le fichier Excel (optionnel)
